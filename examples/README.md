# TaskbarUtil Examples

Example PowerShell scripts for common taskbar setup scenarios.

## Scripts

- **setup-dev-workstation.ps1** - Sets up a developer taskbar with Chrome, Terminal, VS Code, etc.
- **setup-office-workstation.ps1** - Sets up an office taskbar with Edge, Outlook, Teams, etc.

## Usage

```powershell
# Preview what would be done
.\setup-dev-workstation.ps1 -DryRun

# Apply the layout
.\setup-dev-workstation.ps1

# With verbose output
.\setup-dev-workstation.ps1 -Verbose
```
